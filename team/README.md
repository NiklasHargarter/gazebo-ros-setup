# Team / project-internal docs

These files describe the team workflow and coordination around the specific project this repo was first used in. They are **not** part of the published setup guide and are deliberately excluded from the GitHub Pages site in `docs/`.

- [ros2-gazebo-integration.md](ros2-gazebo-integration.md) — interface contract and team integration architecture

If you are just trying to use the container, ignore this folder and read the user docs in [`docs/`](../docs/) instead.
